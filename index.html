多物体运动同多个效果

starmove(_this, {top: -15,opacity: 0},10, function() { _this.style.top = 30 + 'px'; starmove(_this, {top: 15,opacity: 100},10) })

分享

速度缓冲

starmove(this,{left: 0},30)

starmove(this,{left: -200},30)

透明渐变

starmove(this, {opacity:100});

starmove(this, {opacity:30});

链式运动

starmove(this,{opacity:100,width:400},function(){ starmove(this,{height:400}) });

starmove(this,{opacity:30,width:200},function(){ starmove(this,{height:200}) });

多物体链式运动

odiV3[i].onmouseover = function() { starmove(this, {width:400,opacity:100}); }

odiV3[i].onmouseout = function() { starmove(this, {width:200,opacity:30}); }

调用方法：

starmove(dom,{calss_attr_name: parameter},time,callback);

options参数

参数 方法 说明
dom 'this' 要进行操作的dom节点
calss_attr_name {top: -15,opacity: 0} 要进行的动作,多个效果用英文逗号分隔
time 30 动画时间
callback function(){ starmove(odiV4,{height:400}) } 回调函数
